When a SELECT INTO statement causes the predefined exception TOO_MANY_ROWS the values of the variables in the INTO clause are undefined.

Noncompliant Code Example

begin
  select empno
    into var
    from emp;
exception
  when too_many_rows then
    null;
end;

Compliant Solution

begin
  select empno
    into var
    from emp;
exception
  when too_many_rows then
    var := null;
end;
