Quando a cláusula IF de uma estrutura IF-THEN-ELSE não completa normalmente, a cláusa ELSE é desnecessária.

Código em desconformidade

BEGIN
  IF condition THEN
    RETURN value;
  ELSE
    value := NULL;
  END IF;
END;

Código correto

BEGIN
  IF condition THEN
    RETURN value;
  END IF;
  
  value := NULL;
END;
